# pagShrink
jquery plugin for Bootstrap 4: auto shrinks pagination (responsive)

## usage

_activate_

~~~js
$('.pagination').pagShrink();
~~~

_html_

~~~html
<ul class="pagination d-print-none">
	...
	<li class="page-item">
		<a class="page-link" href="#">
			whatever...
		</a>
	</li>
	...
</ul>
~~~
